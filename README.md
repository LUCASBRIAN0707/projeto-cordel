Projeto Cordel
Este projeto foi desenvolvido durante o curso em vídeo de JavaScript. O objetivo foi praticar a manipulação de DOM, eventos e a lógica de programação. O projeto simula um texto no estilo de cordel, com interações e navegações entre as seções.
Tecnologias
JavaScript
HTML
CSS
Como foi feito
Sigo os passos das aulas, implementando a lógica de interação com o usuário e estruturando o layout, usando os conhecimentos adquiridos no curso.
